Coding Event Invite

Description


For this project, I designed an animated invitation for a coding event, using Adobe After Effects. I created a visual style that mimics classic binary code, with a black background and flowing green 1s and 0s to evoke an old-school computer terminal.

To highlight the event details, I introduced red and yellow elements that resemble error or failed code. These colors helped the key information stand out while reinforcing the tech-inspired theme. The animation blends both aesthetic design and function, delivering the invitation in a unique and engaging format that fits the coding context.
